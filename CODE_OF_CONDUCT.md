# 行为准则（Code of Conduct）

本项目采用 [Contributor Covenant](https://www.contributor-covenant.org/) 行为准则。

## 我们的承诺

为了建设开放、友好且无骚扰的社区，我们承诺：无论年龄、体型、残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济状况、国籍、外貌、种族、宗教或性取向，都为所有参与者提供积极的体验。

## 我们的标准

鼓励的行为包括（但不限于）：

- 尊重不同观点与经验
- 接受建设性反馈
- 以社区利益为先
- 友善、同理、专业

不可接受的行为包括（但不限于）：

- 侮辱、挑衅、歧视性言论或人身攻击
- 公共或私下骚扰
- 未经同意发布他人隐私信息
- 其他不适当行为

## 执行责任

项目维护者有权并有责任澄清可接受行为，并对任何不可接受行为采取适当、公正的纠正措施。

## 适用范围

本准则适用于项目空间内，以及代表本项目时在公共空间中的行为。

## 违规处理

如需举报不当行为，请通过仓库维护者联系渠道（例如 GitHub Issue/私信）进行反馈。
