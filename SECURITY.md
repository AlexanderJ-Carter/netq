# 安全策略（Security Policy）

感谢你帮助提升 **netq** 的安全性。

## 支持版本

当前仅维护最新版本（`main` 分支 / 最新发布版）。

## 报告安全漏洞

请不要在公开 Issue 中披露安全问题。

推荐通过以下方式私下报告：

- 使用 GitHub 的私密漏洞报告（Security Advisories）：`https://github.com/AlexanderJ-Carter/netq/security/advisories`

报告内容建议包含：

- 影响范围与复现步骤
- 可能的利用方式（如有）
- 你期望的披露时间线

我们会在确认收到后尽快回复，并在修复后发布公告与版本更新。
